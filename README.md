# KeenKeeper - Friendship Manager

KeenKeeper is a web app to help you nurture your relationships by tracking when you last contacted your friends and scheduling follow-ups.

## Technologies Used
- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- React Hot Toast

## Key Features
1. **Friend Dashboard** – View all friends with status, tags, and contact info.
2. **Interaction Logging** – Log calls, texts, or video calls; automatically updates timeline.
3. **Analytics & Filtering** – See interaction stats via pie chart and filter timeline by type.
